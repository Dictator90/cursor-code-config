# Session Handoff Template

Use this template at the end of a session or during stage handoff.

## Goal
- What exact outcome the next session must reach.

## Completed Work
- What is already done and verified.

## Failed Paths
- Attempts that did not work.
- Why they failed (symptom and suspected cause).

## Current State
- Working / broken / blocked.
- Relevant files and artifacts.

## Next Action
- One deterministic next step.
- Command or check to run first.
